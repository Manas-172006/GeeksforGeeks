# AI BI Dashboard

This is an AI-powered Business Intelligence Dashboard built with Streamlit.

## Features

- Interactive data visualization
- Natural language query processing
- AI-powered insights using Large Language Models
- Chart generation for data analysis

## Project Structure

- `app.py`: Main Streamlit application
- `llm_handler.py`: Handles LLM interactions
- `query_engine.py`: Processes user queries
- `chart_generator.py`: Generates charts and visualizations
- `dataset_loader.py`: Loads and preprocesses datasets
- `data/dataset.csv`: Sample dataset file
- `requirements.txt`: Python dependencies

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `streamlit run app.py`

## Usage

1. Load your dataset by placing it in the `data/` folder
2. Run the app and enter natural language queries
3. View AI-generated insights and visualizations
